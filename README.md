<h1 align='center'><strong>Covid Tracker ⚡</strong></h1>
## **Main technologies 📝**


- Tailwind CSS
- HighCharts
- Axios


## **Install 🔨**

1. Clone Project
  ```sh
  git clone https://github.com/haudnn/Covid_Tracker
  ```
2. Install Packages

```sh
  yarn install | npm install
```
3. Using

```sh
  yarn dev | npm run dev
```
